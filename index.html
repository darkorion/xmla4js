Xmla4js - OLAP for web applications

Welcome to your xmla4js homepage!

Xmla4js is a standalone cross-browser javascript library that provides basic XML/A capabilities, allowing webdevelopers to access data and metadata from OLAP providers for use in rich web applications. XML for Analysis (XML/A) is the defacto standard protocol to communicate with OLAP servers. It defines a SOAP webservice that allows clients to obtain metadata and to execute MDX (multi-dimensional expressions) queries using HTTP as the underlying transport protocol.

To start using xmla4js, please go to the google code project page, and read how to setup xmla4js here: http://code.google.com/p/xmla4js/wiki/InstallingAndUsing. For more general information on XML/A, read the specification (MS Word format). For your convenience, the specification is also available in open document format and portable document format.

After setting up xmla4js on your system, you can start using these samples:

  • discover-synchronous.html - Illustrates how to use Xmla.js to do synchronous retrieval of OLAP metadata. The sample lets you enter a URL for a XML/A webservice, and then allows you to explore what datasources it provides, what catalogs and cubes it contains, and how these are organized, all the way down to the members and their properties.
