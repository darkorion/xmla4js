Class Xmla.Rowset

This class implements an XML/A Rowset object, which is the result of performing the Discover method (see discover()).

Constructor

Xmla.Rowset ( node , requestTtype )
Parameters:
node <DOMDocument> The responseXML result returned by server in response to a Discover request.
requestTtype <string> The requestType identifying the particular schema rowset to construct. This facilitates implementing field getters for a few complex types.

Methods

getFields

fieldDef[] getFields ( )
Retrieve the collection of fieldDef objects for this rowset. A fieldDef describes a field (column). It has the following properties:
name
string. This is the (possibly escaped) name of the field as it appears in the XML document
label
string. This is the human readable name for this field. You should use this name for display purposes and for building restrictions
index
int. The ordinal position of this field. Fields are numbered starting from 0.
type
string. The name of the XML data type for the values that appear in this column
minOccurs
string. The minimal number of occurrences of a value. "0" means the field is optional.
maxOccurs
string. If this is parseable as an integer, that integer specifies the number of times a value can appear in this column. "unbounded" means there is no declared limit.
getter
function. This function is used to extract a value from the XML document for this field.
Returns: fieldDef[]
An (ordered) array of field definition objects.

hasMoreRows

bool hasMoreRows ( )
Indicates wheter the rowset that can be traversed. You can use this method together with the next() method to drive a while loop to traverse all rows in the rowset, like so: 
 while(rowset.hasMoreRows()){
     ...process row...
     rowsete.next();
 }
Returns: bool
true in case there are more rows to traverse. false if all rows have been traversed.
