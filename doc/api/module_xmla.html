Module: xmla

Xmla implements a XML for Analysis (XML/A) client in javascript. Using this utility you can communicate with XML/A enabled OLAP servers to obtain metadata and to issue MDX queries.

This module contains the following classes:

  • Xmla
  • Xmla.Rowset
